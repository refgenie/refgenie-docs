## What's new

<CardGrid>
  <Card title="Database-backed storage" icon="seti:db">
    SQLite by default, PostgreSQL for scale. No more flat YAML config files.
  </Card>
  <Card title="Built-in server" icon="laptop">
    Serve your assets over HTTP without a separate package.
  </Card>
  <Card title="GA4GH identifiers" icon="approve-check">
    Sequence-derived genome identifiers replace arbitrary genome names.
  </Card>
  <Card title="Remote operation" icon="cloud">
    Use `seekr` to access assets on S3 or other cloud storage.
  </Card>
</CardGrid>

